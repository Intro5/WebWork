1. Events are used to handel JS interations on webpages as a result of
    browser and user actions. For example when a user clicks a button.
    1. onclick
      Activated when the user clicks their mouse.
    2. onsubmit
      Activated when the webpage submits a form.
    3. onmouseover
      Activated when the user mouses over the corresponding element.
    4. ondrag
      Activated when the corresponding element is dragged.
    5. ondrop
      Activated when the corresponding element is dropped
    6. oninput
      Activated when the corresponding element recives user input.
